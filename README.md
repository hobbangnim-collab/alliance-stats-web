# Alliance Stats Web

Streamlit web app for uploading alliance CSV files, viewing charts, and downloading an Excel report.

## Files

- `app.py`: Streamlit entrypoint
- `alliance_stats_report.py`: CSV parsing and Excel report logic
- `requirements.txt`: Python dependencies for Streamlit Cloud
- `data/`: optional CSV folder. Put CSV files here if you do not want to upload them every time.
- `members.txt`: optional member filter file. Put one member per line.

## Deploy

Use `app.py` as the Streamlit entrypoint.
